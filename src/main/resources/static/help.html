What is FlickShare about?

FlickShare is an application that allows a groups if users within dirving distance (don't have to be) to share their video collection with friends and family. This is a close systems and by invitation only.

How to use the application

This is a single page application. From the home page, type any part of a movie titles on the search box. If the "Internet" radio button is selected, it will find any movie matching the word type in the search box. If "Local Collection" is selected, the local database will be searched.

Add movies to the collection

After serching the internet for a movie, click on the poster to view its details and press the "Add to Collection" button.

How to add a review

Once a movie is added to the local database, click the "Local Collection" radio and click the Search button. This will list all movies in the local collection. Click on a poster to view the details, type your review in the review box and click Save Review. You can also search by title.
